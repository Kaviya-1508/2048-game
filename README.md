# 🎮 2048 Game

A modern, fully responsive 2048 game built with **React** and **TypeScript**. Play on desktop with arrow keys or on mobile with smooth swipe controls!

## ✨ Features

- 🎯 **Classic 2048 Gameplay** - Merge tiles to reach 2048
- 📱 **Mobile Optimized** - Swipe controls with prevent-refresh
- 💻 **Desktop Ready** - Arrow key controls
- 🎨 **Modern Design** - Beautiful gradients and animations
- 💾 **Local Storage** - Best score saved automatically
- 🚀 **Fast & Responsive** - Works on all screen sizes
- 🎉 **Win/Loss Detection** - Clear victory and game over states
- 🏃 **Splash Screen** - Animated welcome page
- ⏳ **Loading Animation** - Smooth transition to game

## 🎮 How to Play

1. Click **PLAY NOW** on the splash screen
2. Use **arrow keys** (desktop) or **swipe** (mobile) to move tiles
3. Same numbers merge into double value
4. Each merge adds to your score
5. Reach **2048** to win!
6. Game ends when no moves remain

## 📦 Installation

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Clone and Run

```bash
# Clone the repository
git clone https://github.com/Kaviya-1508/2048-game.git

# Navigate to frontend folder
cd 2048-game/frontend

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
